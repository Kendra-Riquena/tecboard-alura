# Projeto Tecboard

Este é um projeto de landing page para a marca "Tecboard", desenvolvido com foco principal nas tecnologias HTML e CSS.

## Destaques do Projeto

- **Estrutura HTML5:** Utilização de tags semânticas para uma melhor estrutura e acessibilidade.
- **Estilização com CSS3:** Foco na criação de um layout moderno e visualmente agradável.
- **Design Adaptativo:** A página foi construída para se adaptar a diferentes tamanhos de tela, garantindo uma boa experiência de usuário tanto em desktops quanto em dispositivos móveis.

O objetivo foi praticar e demonstrar conceitos fundamentais de desenvolvimento web front-end.
